# Turk Dunyosi Ayol Ijodkorlari Platformasi

Turkiy xalqlar ayol ijodkorlarining hayoti, ijodi va media materiallarini jamlovchi raqamli platforma.

## Project Structure

```
anthology_app/
├── mobile_app/    # Flutter mobile application
├── backend/       # Node.js + Express.js REST API
├── docs/          # Technical documentation
└── README.md
```

## Getting Started

### Backend
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

### Mobile App
```bash
cd mobile_app
flutter pub get
flutter run
```
